# khoiucd.github.io

Personal website for Khoi Duc Nguyen.

## Development

```bash
bundle install
bundle exec jekyll serve
```

The local site runs at `http://localhost:4000`.

## Content

Most content lives in `_config.yml` and `_data/`:

- `_data/publications.yml`
- `_data/services.yml`

The homepage is `index.html` and intentionally uses `layout: null`.
